## Getting Started

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. Install dependencies
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server
   ```bash
   npm start
   # or
   yarn start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Deployment

### Netlify

1. Install Netlify CLI
   ```bash
   npm install -g netlify-cli
   ```

2. Login to Netlify
   ```bash
   netlify login
   ```

3. Deploy using the built-in script
   ```bash
   npm run deploy
   ```

### Vercel

1. Install Vercel CLI
   ```bash
   npm install -g vercel
   ```

2. Deploy to Vercel
   ```bash
   vercel
   ```

## Customization

1. Replace placeholder images in the `/public` folder
2. Update personal information in the components
3. Add your own projects and experience
4. Modify colors in `tailwind.config.js` to match your personal brand
